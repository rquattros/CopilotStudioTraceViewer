How do I set this up?
  1. Go to Entra ID > App Registrations and create a new registration
  2. Set Redirect URI to http://localhost (or the URL where you host this page) as type Single-page application (SPA)
  3. Under API Permissions, add Dynamics CRM > user_impersonation (delegated)
  4. Copy the Application (client) ID and Directory (tenant) ID from the Overview page and paste them above
  5. Your user account needs the Bot Transcript Viewer security role in the target environment
